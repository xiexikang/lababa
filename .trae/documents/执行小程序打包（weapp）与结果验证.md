**目标**
- 执行 `npm run build:weapp`，生成微信小程序产物到 `dist/weapp`，并指导你在微信开发者工具运行验证。

**执行步骤**
- 运行打包命令：`npm run build:weapp`
- 确认输出目录：`g:\www-xxk\xcx-lababa\dist\weapp`（已按平台分目录输出）
- 打开微信开发者工具：导入 `dist/weapp` → 设置 `AppID` → 点击“编译”
- 如果未显示最新改动：执行“清缓存并重启”后重新编译

**可选（自动刷新）**
- 启动持续编译：`npm run dev:weapp`，在开发者工具勾选“监视文件变更”“自动编译”

**完成标准**
- 构建成功、目录为 `dist/weapp`，首页与个人中心按钮显示正常，未登录统一跳转登录页，邀请仅在点击后触发。
