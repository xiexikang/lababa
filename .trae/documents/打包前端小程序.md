**目标**

* 生成最新的小程序构建产物到 `dist/`，不影响当前 H5。

**执行步骤**

* 使用构建命令：`npm run build:weapp`。

* 构建完成后，产物位于 `g:\www-xxk\xcx-lababa\dist`。

* 在微信开发者工具导入 `dist`，设置 `AppID` 并点击“编译”。

**验证**

* 首页、个人中心授权与排行榜头像昵称联动正常。

* 若未看到最新改动，执行“清缓存并重启”后再编译。

**说明**

* 此操作不改动 H5；如需实时刷新，可另启 `npm run dev:weapp` 并在开发者工具勾选“监视文件变更”“自动编译”。
